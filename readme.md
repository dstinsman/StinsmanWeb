# stinsman.com — David & Deanna Stinsman

Personal website for David & Deanna Stinsman, a husband-and-wife real estate investment team based in the Tampa Bay area. The site introduces who they are, showcases their work, and connects them with potential lending partners and property sellers.

---

## Pages

### [Stinsman.html] — Home / Landing Page
The main entry point. Features a full-screen hero image with animated scroll arrow, sticky navigation with dark mode toggle, and sections covering About, Services, and Partner opportunities. Includes a call-to-action for property sellers and a contact footer with a background image.

### [projects.html] — Recent Projects
A photo gallery showcasing completed renovation projects. Built with a carousel layout and a lightbox viewer for full-size image browsing with previous/next navigation.

### [pml.html] — Private Money Lending
An interactive, multi-panel timeline walking potential investors through the private lending journey — from initial conversation to deal close. Includes a return calculator where visitors can enter a loan amount, interest rate, and term to see projected earnings.

### [buybox.html] — Buy Box
Outlines the types of properties David & Deanna are actively looking to purchase. Helps sellers quickly determine if their property might be a fit. Built with a creative flair to resemble a vintage home's architectural blueprint.

---

## Features

- **Dark mode** — toggle persists across pages via [localStorage]
- **Responsive design** — mobile-friendly layouts across all pages
- **Lightbox image viewer** — full-screen photo browsing on the Projects page
- **Return calculator** — interactive tool on the Private Lending page
- **Sticky navigation** — consistent nav with smooth hover animations across all pages
- **CSS custom properties** — theme variables used throughout for consistent, maintainable styling

---

## Tech Stack

- Plain HTML5, CSS3, and vanilla JavaScript — no frameworks or build tools required
- [Google Fonts](https://fonts.google.com/) — Inter
- [Font Awesome 6](https://fontawesome.com/) — icons
- Custom [.webp]background images
